# Spotiplay
This is a full-stack music streaming website equipped with user authentication and online transaction/subscription, inspired by Spotify.

Techstack:
- Typescript
- Next.js 13
- TailwindCSS
- PostgresSQL
- Supabase 
- Stripe (To be Added)
